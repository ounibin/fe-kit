# getUrlParam(url: string, name: string): string

从 url 中获取指定参数的值

## 参数

| 参数                | 类型       | 默认值 | 说明                                                                     |
| ------------------ | --------   | --------  |------------------------------------------------------------------------ |
| url                | `string`   |   |  要打开的链接                                      |
| name            | `string`   |    | 要获取的参数名                                      |              




## 返回值

`string` 指定参数的值

## 示例

```js
getUrlParam('https://baidu.com?q=你好', 'q') // => 你好
```
