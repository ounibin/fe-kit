# number2price(num: number, decimal = 2):string

将数字转换为价格字符串，默认保留两位小数

## 参数

| 参数                | 类型       | 默认值 | 说明                                                                     |
| ------------------ | --------   | --------  |------------------------------------------------------------------------ |
| num                | `number`   |    |  需要被转化的数字                                         |
| decimal            | `number`   |  2 | 精度，小数点后的位数                                      |              




## 返回值

`string` 指定参数的值

## 示例

```js
  number2price(1.5, 0) // => '2'
  number2price(1.2, 0) // => '1'
  number2price(1.2, 1) // => '1.2'
  number2price(1.2, 2) // => '1.20'
```
